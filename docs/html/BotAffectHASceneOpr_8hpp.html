Detailed Description

Operation that commands Home Automation system to turn scenes (primarily lighting) on/off.

In a HomeBot system, the Home Automation system is responsible for turning scenes on/off. This operation provides a way for a HomeBot service robot to turn scenes on/off through the Home Automation system as part of a HomeBot behavior.
