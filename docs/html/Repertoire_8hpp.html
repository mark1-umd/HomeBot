Detailed Description

A HomeBot repertoire is a collection of behaviors for a HomeBot service robot.

Date
May 9, 2017 - Creation

HomeBot represents a behavior as collections of operations that, when executed together, produce activity that is described as the service robot carrying out the behavior. Since it is useful for a service robot to possibly have more than one behavior, a collection of behaviors must be maintained for each service robot in a HomeBot system. That collection is termed a repertoire, and is maintained by this class. The repertoire protects its behaviors, only providing a copy of the behaviors when requested, not a reference or pointer to the Repertoire's stored behavior.
