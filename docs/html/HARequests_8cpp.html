[Deprecated] Translate Home Automation service requests from the HomeBot system into Home Automation commands

HomeBot entities that need services provided by an external Home Automation system that has been interfaced to the HomeBot system use the ROS Action Protocol to request the Home Automation services defined as ROS actions. This ROS node provides the various action servers required for this purpose.
